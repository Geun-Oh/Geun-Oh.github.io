### vim can't open file for writing

vim src/file.text 와 같이 디렉토리를 포함한 경로에서 vim 을 실행할 때, src/ 폴더가 존재하지 않으면 (경로 접근 불가 시) 에러 발생함.
단순한 에러들은 반복해서 까먹지 말기.

### git add -p

는 untracted file에는 동작하지 않음. 추적할 변경사항이 없어서.
처음 git에 올릴 때는 파일 단위로 올리자.
